# 📂 File Explorer - Bash Script

## 📝 Overview
This is a lightweight file explorer developed in **Bash** for the **Operating Systems** course at **UNIFEI**. It provides a command-line interface to navigate directories and perform basic file system operations.

## 🛠️ Features
* **Navigation:** Browse through local directories.
* **File Management:** List, view, and organize files.
* **Automation:** Scripted shortcuts for common terminal tasks.
